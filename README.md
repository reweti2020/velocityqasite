# VelocityQA Website

This repository contains the website for VelocityQA, a professional quality assurance service for startups and small businesses.

## Development Environment

The website is being built in stages with core functionality deployed incrementally.

## Branches:

- `main`: Production-ready code
- `UAT-DEV`: User acceptance testing and development
- `feature-*`: Feature branches for specific functionality

## Components:

- Quality Assurance Toolkit
- LinkedIn Post Library
- Resource Center
- Contact Forms
- Service Descriptions
