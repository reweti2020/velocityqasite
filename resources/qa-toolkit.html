QA Checklists

Mobile App Testing Checklist

A comprehensive checklist for testing mobile applications across iOS and Android platforms, covering functionality, usability, and performance.

Mobile iOS Android 
# MOBILE APP TESTING CHECKLIST

## INSTALLATION & SETUP
- [ ] App installs correctly on clean device
- [ ] App installs as update over previous version
- [ ] App can be uninstalled properly
- [ ] Installation size is within acceptable limits
- [ ] App permissions are appropriate and clearly explained
- [ ] App appears properly in app drawer/home screen
- [ ] App icon displays correctly in all required resolutions

## FUNCTIONALITY
- [ ] All features work according to specifications
- [ ] Required user flows complete successfully
- [ ] All buttons and controls function properly
- [ ] Text fields handle input correctly (including validation)
- [ ] Error messages are clear and helpful
- [ ] Push notifications display correctly and take user to appropriate screen when tapped
- [ ] Data is saved correctly when app is closed
- [ ] Data synchronizes properly with backend services

Web Accessibility Testing Checklist

A detailed WCAG 2.1 compliance checklist for ensuring web applications meet accessibility standards.

Accessibility WCAG Compliance 
# WEB ACCESSIBILITY TESTING CHECKLIST (WCAG 2.1)

## PERCEIVABLE

### Text Alternatives
- [ ] **1.1.1 Non-text Content (A)** - All images have appropriate alt text
- [ ] Decorative images have empty alt attributes or are CSS backgrounds
- [ ] Complex images (charts, diagrams) have extended descriptions
- [ ] Form input fields have associated text labels
- [ ] Audio and video have text alternatives (transcripts, captions)

### Time-based Media
- [ ] **1.2.1 Audio-only and Video-only (A)** - Pre-recorded audio-only and video-only content has alternatives
- [ ] **1.2.2 Captions (A)** - Captions provided for all pre-recorded audio
- [ ] **1.2.3 Audio Description or Media Alternative (A)** - Audio description or alternative provided for video
- [ ] **1.2.4 Captions (Live) (AA)** - Captions provided for live audio content
- [ ] **1.2.5 Audio Description (AA)** - Audio description provided for all pre-recorded video
